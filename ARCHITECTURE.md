# ARCHITECTURE.md — 스타일씨(STYLE C) 시스템 아키텍처

## 1. 프로젝트 개요

스타일씨(STYLE C)는 광고주(브랜드/사업자)와 인플루언서를 연결하는 **구독형 인플루언서 체험단 마케팅 플랫폼**입니다.

- 운영사: 주식회사 스타일씨 코퍼레이션
- 서비스 URL: https://www.stylec.co.kr
- 플랫폼: 웹(PC/Mobile) + 앱(iOS, Android)

## 3. 주요 도메인 및 역할

### 3-1. 광고주 도메인 (Advertiser Domain)
- 체험단 캠페인 등록 및 관리
- 인플루언서 선택 및 승인
- 리뷰 결과 레포트 조회
- 구독 플랜 결제 관리

### 3-2. 인플루언서 도메인 (Influencer Domain)
- 체험단 탐색 및 신청
- 리뷰 제출 및 인증
- 포인트/페이백 정산
- 프로필 관리 (SNS 채널, 팔로워 수, 일방문자 수)

### 3-3. 캠페인 도메인 (Campaign Domain)
- 체험단 유형 관리 (페이백+구매평, 구매평, 제공형, 방문형, 기자단, 미션형)
- 지원 플랫폼 (블로그, 인스타그램, 인스타릴스, 스마트스토어, 쿠팡, 배달앱)
- 모집 인원 및 기간 관리
- 콘텐츠 승인 워크플로우

### 3-4. 정산 도메인 (Settlement Domain)
- 구독 요금 정기결제
- 인플루언서 페이백 처리
- 정산 내역 관리

## 5. 백엔드 서비스 상세

### Auth Service
- 광고주 / 인플루언서 역할 기반 인증
- JWT 토큰 기반 세션 관리
- SNS 소셜 로그인 (네이버, 카카오)

### Campaign Service
- 체험단 CRUD
- 신청자 목록 관리 및 당첨자 선정
- 체험단 상태 관리 (모집중 → 진행중 → 완료)
- 캠페인 결과 레포트 생성

### Influencer Service
- 인플루언서 프로필 관리
- SNS 채널 연동 (팔로워 수, 방문자 수 자동 수집)
- 인플루언서 등급 관리

### Review Service
- 리뷰 콘텐츠 제출 및 검수
- SNS 콘텐츠 조회수 트래킹
- 콘텐츠 승인/반려 워크플로우

### Payment Service
- 구독 플랜 정기결제 (PG 연동)
- 인플루언서 포인트/페이백 정산
- 결제 내역 및 세금계산서 관리

### Notification Service
- 이메일 / SMS / 앱 푸시 알림
- 체험단 모집 알림, 당첨 알림, 리뷰 마감 알림

## 6. 인프라

| 항목 | 내용 |
|------|------|
| 클라우드 | AWS (추정) |
| 웹 서버 | Nginx |
| 앱 서버 | Node.js 또는 Java Spring Boot |
| 데이터베이스 | MySQL (주 DB), Redis (캐시/세션) |

## 7. 외부 연동

| 서비스 | 용도 |
|--------|------|
| 채널톡 (Channel.io) | 실시간 채팅 상담 |
| 네이버 블로그 API | 블로그 방문자 수, 콘텐츠 확인 |
| 인스타그램 Graph API | 팔로워 수, 게시물 조회수 |
| PG사 (결제) | 정기구독 결제 처리 |
| 공정거래위원회 API | 사업자 정보 확인 |
| App Store / Google Play | 모바일 앱 배포 |
