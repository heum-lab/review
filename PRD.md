## 1. 제품 목적 (Product Purpose)

스타일씨는 **광고주(브랜드/사업자)가 월 정액 구독료로 인플루언서 체험단을 무제한 운영**할 수 있는 마케팅 플랫폼입니다. 복잡하고 비용이 큰 인플루언서 마케팅 과정을 자동화하여 5분 내 체험단 등록, 직접 인플루언서 선택, 실시간 결과 레포트 제공을 목표로 합니다.

---

## 2. 배경 및 문제 정의

### 기존 인플루언서 마케팅의 문제점
- 내부 운영 시 인플루언서 서치 → 연락 → 협의 → 회신 대기의 과정에서 최소 3일 이상 소요
- 건당 10~20만원의 높은 비용 + 담당 인력 인건비 추가 발생
- 인플루언서 신뢰도, 성실도 검증의 어려움
- 리뷰 결과 추적 및 콘텐츠 수집의 불편함

### 스타일씨의 해결책
- 단 5분 내 체험단 등록 완료
- 월 30만원 정액제로 무제한 체험단 운영 (건당 약 1,500~3,000원 수준)
- 광고주가 직접 SNS 지표를 확인하고 인플루언서 선택
- 실시간 리뷰 및 결과 레포트 자동 제공

---

## 3. 주요 사용자 (Users)

### 3-1. 광고주 (Advertiser)
- **정의:** 인플루언서 마케팅을 원하는 브랜드, 쇼핑몰, 음식점, 스마트스토어 셀러, 쿠팡 셀러
- **핵심 니즈:** 저비용으로 많은 리뷰 생성, 인플루언서 직접 선택, 결과 수치 확인
- **페인 포인트:** 높은 비용, 긴 진행 시간, 낮은 인플루언서 신뢰도

### 3-2. 인플루언서 (Influencer)
- **정의:** 블로그, 인스타그램, 유튜브, 네이버 인플루언서 등록 창작자
- **핵심 니즈:** 다양한 체험단 참여를 통한 수익 창출 및 콘텐츠 소재 확보
- **페인 포인트:** 체험단 탐색 및 신청 불편, 정산 지연

## 4. 핵심 기능 요구사항 (Core Features)

### 4-1. 광고주 기능

#### F-A-01: 체험단 등록 (5분 내 완료)
- 체험단 유형 선택 (페이백+구매평, 구매평, 제공형, 방문형, 기자단, 미션형)
- 지원 플랫폼 선택 (블로그, 인스타그램, 인스타릴스, 스마트스토어, 쿠팡, 배달앱)
- 모집 인원, 기간, 제공 내용 설정
- 체험단 이미지 및 가이드라인 등록

#### F-A-02: 인플루언서 직접 선택
- SNS 채널별 팔로워 수 확인
- 블로그 일방문자 수 확인
- 인플루언서 성실도/활동도 지표 확인
- 신청자 목록에서 원하는 인플루언서 선택 및 승인

#### F-A-03: 실시간 결과 레포트
- 진행중인 체험단 현황 대시보드
- SNS 콘텐츠 링크 및 조회수 자동 수집
- 리뷰 승인/반려 처리
- 기간별 누적 콘텐츠 현황

#### F-A-04: 구독 플랜 관리
- 프리미엄 플랜 구독 (월 30만원, VAT 별도)
- 정기결제 카드 등록 및 변경
- 결제 내역 및 세금계산서 발급

#### F-A-05: 광고주 센터
- 진행중 캠페인 전체 관리
- 인플루언서 소통 채널
- 공지사항 및 가이드 확인

---

### 4-2. 인플루언서 기능

#### F-I-01: 체험단 탐색 및 신청
- 카테고리/플랫폼별 체험단 목록 조회
- 체험단 상세 정보 확인 (제품, 조건, 보상)
- 원클릭 신청

#### F-I-02: 리뷰 제출 및 인증
- SNS 리뷰 작성 후 URL 제출
- 구매 인증 (페이백형: 구매 영수증 업로드)
- 미션 완료 스크린샷 인증

#### F-I-03: 정산 및 포인트 관리
- 페이백 금액 확인 및 수령
- 포인트 적립 내역 조회
- 출금 신청

#### F-I-04: 프로필 관리
- SNS 채널 연동 (블로그, 인스타그램, 유튜브 등)
- 팔로워/방문자 수 자동 갱신

## 5. 체험단 유형 상세 요구사항

| 유형 | 프로세스 | 지원 플랫폼 | 보상 방식 |
|------|----------|-------------|-----------|
| 페이백+구매평 | 선구매 → SNS리뷰 + 구매평 작성 | 블로그, 인스타릴스, 인스타그램, 스마트스토어 + 구매평 | 구매금액 100% 페이백 |
| 구매평 | 선구매 → 구매처 구매평 작성 | 구매평 전용 | 구매금액 100% 페이백 |
| 제공형 | 제품 수령 → SNS 리뷰 작성 | 블로그, 유튜브, 인스타그램 등 | 제품 제공 |
| 방문형 | 장소 방문 체험 → SNS 리뷰 | 블로그, 유튜브, 인스타그램 등 | 방문 비용 지원 |
| 기자단 | 자료 수령 → SNS 리뷰 작성 → 승인 | 블로그, 인스타그램 등 | 원고료 지급 |
| 미션 | 미션 수행 → 스크린샷 인증 | 전체 | 완료 보상 지급 |

## 6. 비기능 요구사항 (Non-Functional Requirements)

### 6-1. 성능
- 체험단 등록 완료: 5분 이내
- 체험단 페이지 로딩: 3초 이내
- 월 방문자 150만명 이상 처리 가능한 확장성

### 6-2. 가용성
- 서비스 가동률 99.9% 이상
- 고객센터 운영: 평일 10:00~18:00 (점심 12:30~13:30 제외, 주말·공휴일 제외)
- 채팅 상담: 채널톡을 통한 비실시간 문의 접수

### 6-3. 보안
- 개인정보보호법 준수
- 광고주 문의 정보 수집 후 90일 이내 파기
- HTTPS 필수 적용

### 6-4. 플랫폼 지원
- 웹: PC 및 모바일 브라우저 (반응형)
- 앱: iOS 및 Android 네이티브/웹뷰 앱

## 9. 로드맵 (Roadmap)

### Phase 1 (현재 운영 중)
- 광고주 랜딩 페이지 및 문의 접수
- 인플루언서 체험단 신청 플랫폼
- 광고주 센터 (실시간 결과 레포트)
- iOS/Android 앱

### Phase 2 (확장)
- 배달 전문 음식점 특화 서비스 (/trials/about/bedal)
- 추가 플랫폼 지원 (인스타릴스, 쿠팡이츠 등)

### Phase 3 (미래)
- AI 기반 인플루언서 매칭 자동화
- 글로벌 시장 진출
- 데이터 분석 고도화

---

## 10. 제약 사항 (Constraints)

- 사업자당 1개 브랜드만 이용 가능 (다수 브랜드 운영 시 별도 계약 필요)
- VAT는 구독료에 별도 적용
- 신규 고객 50% 할인 혜택은 변경/종료 가능
- 고객센터는 평일 업무 시간에만 운영
